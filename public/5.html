Guide to learning Frontend Web Development - part 1




How do you start? Where do you start?

Before start, create a github account and get comfortable with github. You will push your work to github everyday. Employers will look at your github profile and judge you according to how active you are on github.



Phase 1: HTML, CSS and JAVASCRIPT

START WITH HTML AND CSS (duration: 1 month)


- HTML : It’s the content
(text, images, videos, links)

- CSS : It’s the style
(layouts, colors, typography, animations)



  • Learn the basics

  • Practice building some websites (google, facebook, 42intra)

  • Learn how to make 1 column layouts, 2 column layouts, 3 ...

  • Learn about Float and Clear, Flexbox and Grid



  • Learn about Responsive Designs (how website look on phone or tablet compared to desktop)

  • Learn about Semantic approach of writing HTML5 and CSS3 (html tags, header, nav, ...)

  • Learn about Media Queries (make website responsive depending on devices)

  • Learn about typography

  • Build some more projects and use everything you learnt till now.

  • Practice, practice, practice







Learn JAVASCRIPT (duration 2 weeks)

  • Start with basics (variables, types, if, loops, functions, objects)

  • Solve coding challenges (solve 20 and then after that 1 a day) => link to coding challenge + solution : consistency is key

  • Learn object oriented programming Basics (prototype, this, instantiation, inheritance)

  • Learn the Basics, you will forget, but it’s gonna be easier to learn again when u need it

  • Learn why OOP is important






LEARN DOM - Vanilla-js

At this point do not use any library. Many job interviews require you to know to code in vanillaJs in their code test.


  • Basics (querying, common methods)

  • Start building things (simple js clock, countdown timer, rock paper scissors, todolist)


Note:

At this point, you should be comfortable with html and css and you should be decent with javascript. So you can start making money with what you have learnt so far
